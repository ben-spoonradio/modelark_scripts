## ⚙️ 내 마음대로 설정하기 (config.txt)

동영상을 내 취향에 맞게 만들 수 있어요! `config.txt` 파일을 수정하면 됩니다.

```ini
# 동영상 화질 (선택: 480p, 720p, 1080p)
# ⚠️ 1080p는 Pro 모델에서만 지원
resolution=720p

# Pro 모델 사용 (true/false)
# Pro 모델: 더 높은 품질, 1080p 지원, 더 많은 토큰 사용
# Lite 모델: 표준 품질, 빠른 생성, 적은 토큰 사용
use_pro_model=true

# 동영상 비율 (선택: 16:9, 9:16, 1:1, 4:3, 21:9, 9:21)
# 📱 세로 동영상(인스타/틱톡): 9:16
# 📺 가로 동영상(유튜브): 16:9  
# 📷 정사각형(인스타 피드): 1:1
# ⚠️ 이미지-to-비디오: adaptive, keep_ratio만 지원
ratio=9:16

# 동영상 길이 (5초 또는 10초)
duration=5

# 부드러움 정도 (16 또는 24)
# 24가 더 부드럽고 영화 같아요
fps=24

# 워터마크 표시 (true/false)
# false로 하면 깔끔해요
watermark=false

# 랜덤 시드 (-1은 매번 다르게)
seed=-1

# 카메라 고정 (true/false)
# true로 하면 흔들림이 적어요
camerafixed=false
```

## 🛠️ 기술적 특징

### 🚀 모델 시스템
- **Pro 모델** (`seedance-1-0-pro-250528`): 최고 품질, 1080p 지원, 모든 기능
- **Lite 모델** (`seedance-1-0-lite`): 빠른 생성, 표준 품질, 경제적

### 🔄 자동 모델 선택
- **텍스트만**: t2v 모델 (모든 화면비 지원)
- **이미지+텍스트**: i2v 모델 (adaptive, keep_ratio만 지원)
- **자동 전환**: 설정에 따라 Pro/Lite 모델 자동 선택

### 📸 이미지 처리
- **지원 형식**: JPEG, PNG, WEBP, BMP, TIFF, GIF
- **자동 변환**: 모든 형식을 JPEG로 자동 변환 (API 호환성)
- **크기 제한**: 최대 10MB, 300px~6000px
- **자동 최적화**: Base64 인코딩을 위한 크기/품질 조정

### 🔧 프레임 추출 기술
- **OpenCV 활용**: 동영상의 마지막 프레임을 고품질로 추출
- **자동 리사이즈**: 1280px 이하로 최적화
- **Base64 인코딩**: 다음 클립에서 바로 사용 가능

### 🎬 동영상 합치기
- **ffmpeg 기반**: 무손실 고품질 동영상 합치기
- **자동 감지**: ffmpeg 설치 상태 자동 확인 및 안내
- **진행률 표시**: 실시간 합치기 진행 상황 표시
- **스마트 처리**: 파일 존재 확인 및 오류 복구

## 🆘 문제가 생겼을 때 (걱정 마세요!)

### 💡 가장 흔한 문제들과 해결법

#### 🔑 "API 키 오류"가 뜰 때
**문제:** API 키가 없거나 잘못 입력했어요
**해결법:** 
1. BytePlus 웹사이트에서 API 키를 다시 확인하세요
2. `./run.sh`를 다시 실행하고 API 키를 정확히 입력하세요

#### 📦 "모듈을 찾을 수 없습니다"라고 뜰 때
**문제:** 필요한 프로그램이 설치되지 않았어요
**해결법:** 
```bash
# 이 명령어로 다시 설치하세요
./run.sh
```

#### 🖼️ "이미지 파일 오류"가 뜰 때
**문제:** 이미지 파일이 너무 크거나 형식이 맞지 않아요
**해결법:**
- 이미지 크기를 10MB 이하로 줄이세요
- JPG, PNG 형식만 사용하세요

#### ⚡ "동영상 생성 실패"가 뜰 때
**문제:** 설정이 맞지 않아요
**해결법:**
- 이미지를 사용할 때는 config.txt에서 `ratio=adaptive`로 설정하세요
- 텍스트만 사용할 때는 어떤 비율이든 괜찮아요
- 1080p 사용 시 `use_pro_model=true`로 설정하세요

#### 🎬 "동영상 합치기 실패"가 뜰 때
**문제:** ffmpeg가 없거나 동영상 파일에 문제가 있어요
**해결법:**
- **macOS**: `brew install ffmpeg`
- **Ubuntu/Debian**: `sudo apt install ffmpeg`
- **Windows**: ffmpeg 공식 사이트에서 다운로드
- 개별 동영상 파일들이 정상적으로 생성되었는지 확인하세요

### 🔧 문제 해결 3단계

#### 1단계: 다시 설치해보기
```bash
./run.sh
```
대부분의 문제가 이것으로 해결돼요!

#### 2단계: API 키 다시 확인하기
- BytePlus 웹사이트에서 API 키가 맞는지 확인
- 복사할 때 앞뒤 공백이 없는지 확인

#### 3단계: 설정 파일 확인하기
- `config.txt` 파일을 열어서 설정이 올바른지 확인
- 이미지 사용 시 `ratio=adaptive` 설정 확인
